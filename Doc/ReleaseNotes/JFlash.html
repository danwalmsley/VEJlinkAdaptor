New features

Version 5.00 (2015-06-08)

  1. DLL/J-Flash: Added flash programming support for Atmel ATSAMR21x19 series.
  2. DLL/J-Flash: Added flash programming support for Freescale MK80FN256 devices.
  3. DLL/J-Flash: Added flash programming support for Nuvoton NUC131LC2AE, NUC131LD2AE, NUC131SC2AE and NUC131SD2AE series devices.
  4. DLL/J-Flash: Added flash programming support for ST STM32F746xE and STM32F746xG series devices.
  5. DLL/J-Flash: Added flash programming support for Toshiba TZ1000 and TZ1001 devices.
  6. J-Flash SPI CL: New product added (Windows, Linux).
  7. J-Flash SPI: New product added (Windows).
  8. J-Flash: Added an option to the project settings to override the default program and erase timeouts.
  9. J-Flash: Added command line options "-usb" and "-ip" to allow temporary override of the J-Link connection settings via command line, for the current session.
  10. J-Flash: Added flash programming support for Cypress CY8C4013 and CY8C4014 series devices.
  11. J-Flash: Added new commands to init steps to merge target data into the image to be flashed. Var Write File 8bit, Var Write File 16bit, Var Write File 32bit
  12. J-Flash: Added sample projects for Toshiba TZ100x series.
  13. J-Flash: Added target voltage information (VTref) to the LOG window on connect.
  14. J-Flash: Added user row flash programming support for Atmel SAMD and SAMR series.
  15. J-Link / J-Flash: Improved flash programming support (especially erase speed in J-Flash) for STM32L072/073 series devices.
